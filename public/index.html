Welcome to the Emoji API Server! :ta-da:

This API was created at the Code Chrysalis Intensive Bootcamp in July of 2018 by TsubasaK111.

By default, you can access all the custom emoji used by Cohort 5 (aka CC5 aka House Dustin aka House Justin aka House Tsuyoshi). It is fully RESTful :smile:. It is auth-less with no hard limits to the number of requests for your convenience, don't abuse it! :shrugs:

If you set up your own server, you can configure it to hook up to any Slack Organization that you are a member to. I'd recommend you play with this instance for the time being, but you can set up your own server here.

Here's what you can do with the API:

List all Emojis

GET localhost:3000/api/emojis using curl or your browser to list all emojis.

Example:

> curl GET localhost:3000/api/emojis
  [ 
   {
   "id": 198,
   "name": "glitch_crab",
   "uri": "https://emoji.slack-edge.com/TBAST357H/glitch_crab/db049f1f9c.png"
   },
   {
   "id": 203,
   "name": "pride",
   "uri": "https://emoji.slack-edge.com/TBAST357H/pride/56b1bd3388.png"
   }
  ]

Show all Information related to an emoji

GET localhost:3000/api/emojis/:id to get detailed information about an emoji, including custom tags. You can get the id of an emoji by first listing all of them with the ~/api/emojis/ request explained above.

Example:

> curl localhost:3000/api/emojis/203 -XGET
  
   {
   "id": 203,
   "name": "pride",
   "uri": "https://emoji.slack-edge.com/TBAST357H/pride/56b1bd3388.png",
   "tags": ["lgbt", "flag", "rainbow"],
   "createdAt": "2018-07-31T15:55:53.322Z" 
   }

Add an emoji

In addition to the custom emojis seededfrom your Slackorganization, you can add additional emoji that you would want to manage in the server with POST localhost:3000/api/emojis/. The body must be in JSON format.

Example:

We're not using curl from this example because it gets too complicated. We highly recommend you use an API interaction tool like Insomnia for venturing around this API.


  // Request:
  POST localhost:3001/api/emojis
  body: {
   "name": "nooice",
   "uri": "https://media.tenor.com/images/bcb961b67dc5ec34381372494c85c8fe/tenor.gif"
  }
  
  // Response:
  {
   "id": 292,
   "name": "nooice",
   "uri": "https://media.tenor.com/images/bcb961b67dc5ec34381372494c85c8fe/tenor.gif"
   "createdAt": "2018-07-31T15:55:53.322Z"
  }

Delete an emoji

Whoops, want to remove an embarassing mistake? It's simple, just DELETE localhost:3001/api/emojis/:id ! Once again, you'll need the emojiId appended to the end of the url to specify the emoji to delete.

Example:

// Request (goodbye piggy emoji!)
  DELETE localhost:3001/api/emojis/9
  
  // Response (id 199 [piggy emoji] is now gone)
  [
   {
   "id": 198,
   "name": "glitch_crab",
   "uri": "https://emoji.slack-edge.com/TBAST357H/glitch_crab/db049f1f9c.png"
   },
   {
   "id": 200,
   "name": "cubimal_chick",
   "uri": "https://emoji.slack-edge.com/TBAST357H/cubimal_chick/85961c43d7.png"
   }
  ]

Modify an Emoji

Want to change the name, url, tags etc of an emoji? It's time for PATCH localhost:3001/api/emojis/:id ! In the request body, include only the properties you wish to modify in JSON format.


  // Request:
  PATCH localhost:3001/api/emojis/200
  body: {
   "name": "yellow_chick",
   "uri": "https://s17-us2.startpage.com/cgi-bin/serveimage?url=https%3A%2F%2Fpixnio.com%2Ffree-images%2F2017%2F02%2F08%2F2017-05-08-07-27-36.jpg&sp=d3d9d1dd0da5b9afecd11b7762f678c8"
  }
  
  // Response:
  {
   "name": "yellow_chick",
   "uri": "https://s17-us2.startpage.com/cgi-bin/serveimage?url=https%3A%2F%2Fpixnio.com%2Ffree-images%2F2017%2F02%2F08%2F2017-05-08-07-27-36.jpg&sp=d3d9d1dd0da5b9afecd11b7762f678c8https://media.tenor.com/images/bcb961b67dc5ec34381372494c85c8fe/tenor.gif",
   "tags": ["cute", "chick", "yellow"]
   "createdAt": "2018-07-31T15:55:53.322Z"
  }

Add tags

Using the patch request, you can also add tags to your emojis! Any new tags will be appended to preexisting tags.

Example: 

// Request:
  PATCH localhost:3001/api/emojis/200
  body: {
   "tags": ["trolling"] 
  }
  
  
  
  // Response:
  {
   "name": "yellow_chicknooicechick",
   "uri": "https://s17-us2.startpage.com/cgi-bin/serveimage?url=https%3A%2F%2Fpixnio.com%2Ffree-images%2F2017%2F02%2F08%2F2017-05-08-07-27-36.jpg&sp=d3d9d1dd0da5b9afecd11b7762f678c8https://media.tenor.com/images/bcb961b67dc5ec34381372494c85c8fe/tenor.gif",
   "tags": ["cute", "chick", "yellow", "trolling"]
   "createdAt": "2018-07-31T15:55:53.322Z"
  }
